# My Programming Projects

A beautiful portfolio website showcasing my programming projects with code viewer.

## Features

- 📊 **Grid & List View** - Browse all programs in grid or list format
- 👁️ **Code Viewer** - View the source code of any program
- 📋 **Copy to Clipboard** - Easily copy code with one click
- 🚀 **Live Demo** - Open any program in a new window
- 📱 **Responsive Design** - Works on desktop and mobile
- 🎨 **Modern UI** - Beautiful gradient design with smooth animations

## How to Use

1. Click on any program button to select it
2. Click "View Code" to see the source code
3. Click "Copy Code" to copy the code to clipboard
4. Click "Open Program" to run the program in a new window

## Deployment to Vercel

### Option 1: Using Git (Recommended)

1. Initialize a git repository:
   ```bash
   git init
   git add .
   git commit -m "Add programs portfolio"
   ```

2. Push to GitHub:
   - Create a new repository on GitHub
   - Add the remote and push your code

3. Deploy on Vercel:
   - Go to [vercel.com](https://vercel.com)
   - Click "New Project"
   - Import your GitHub repository
   - Click "Deploy"

### Option 2: Using Vercel CLI

1. Install Vercel CLI:
   ```bash
   npm install -g vercel
   ```

2. Deploy:
   ```bash
   vercel
   ```

3. Follow the prompts to deploy your project
